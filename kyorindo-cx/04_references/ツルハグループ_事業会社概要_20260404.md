## 杏林堂ロードマップへの示唆

| 視点 | 内容 |
|------|------|
| グループ内ポジション | 杏林堂は静岡県特化。各社が地域ごとに独立運営しており、グループ標準化と独自性のバランスが課題 |
| TGNとの比較 | 売上1,260億円・複数ブランド統合運営のTGNは、杏林堂（静岡）との統合・連携議論の参照例になりうる |
| レデイの完全子会社化（2025/12） | 独自システム多数（D3・RetailCRM）を持つレデイがグループに完全統合。杏林堂固有システムの扱いを考える参考事例 |
| D11の勤怠（勤次郎） | D11はツルハHD標準（T-Net勤怠）ではなく勤次郎を利用。杏林堂（OBIC）と同様に独自勤怠を維持しているケース |
| **TGMDへの出向** | 杏林堂から寺西正芳氏がTGMD食品MD本部長に出向中。グループ内での人事交流・影響力を持つポジション |
