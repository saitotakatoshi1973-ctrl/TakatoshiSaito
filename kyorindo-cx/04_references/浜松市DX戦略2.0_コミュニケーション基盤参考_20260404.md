## 計画概要

| 項目 | 内容 |
|------|------|
| 計画名 | 浜松市DX推進計画2.0（Digital Smart City HAMAMATSU） |
| 計画期間 | 2025年4月〜2030年3月 |
| 目指す方向性 | **「行かない、待たない、書かない」人に寄り添ったデジタル活用** |

### 基本的考え方（5原則）
1. **人が中心、デジタルは手段**
2. 個人情報の保護とプライバシーへの配慮
3. 変化を先取りし、新たな価値を創出する（組織・制度・職員意識の変革）
4. **小さく始めて、改善を繰り返す**
5. **システム所有からサービス利用へ**（クラウド基本）

## コミュニケーション基盤・デジタルワークスタイルに関する施策

### ① デジタルワークスタイル変革（P23）★最重要参考

| 取組 | 内容 | スケジュール |
|------|------|------------|
| **端末のモバイル化** | 端末の無線化・仮想ブラウザ等で場所を選ばない働き方のICTインフラ整備 | 2025年：庁内NW更新 → 2026〜：端末無線化 |
| **フレキシブルな職場環境** | フリーアドレス対応拡充、**クラウドPBX導入で電話のモバイル化** | 2026年：職場モデル設定・希望調査 |
| **テレワーク等の柔軟な働き方** | テレワーク制度化、フリーアドレス指針改定 | 2026年：制度化・指針改定 |

**KPI**: 職場環境にかかる職員満足度 **80%**（2029年度）

### ② 電話DXの推進（P10）

| 取組 | 内容 | スケジュール |
|------|------|------------|
| **クラウドPBX導入** | 現行オンプレPBXをクラウド化。スマートフォンで場所を問わず電話対応 | 2025年：実証実験 → 2026年：基本設計 → 2027年〜：段階移行 |
| 通話可視化・分析システム | 通話内容の可視化・分析、スマートフォン対応ツール | 2025年：実証 → 2026年：構築 → 2027年：稼働 |

### ③ AIの利用推進（P17）

| 取組 | 内容 | KPI（利用職員数） |
|------|------|----------------|
| 職員向け生成AI利用推進 | 生成AIの普及・促進、先進AI導入検証 | 2025年：1,200人 → 2029年：2,000人 |
| AIデジタルツール活用拡大 | AI議事録、電話相談のリアルタイムAIテキスト化、**AIシフト作成ツール** | 順次試験導入 |

### ④ デジタル変革人材の育成・確保（P22）

| 区分 | 内容 |
|------|------|
| **DXコアメンバー** | 10名を公募・選抜し集中育成（デジスマ課主導） |
| **DXメンター** | 100名を育成（5年間で計100名）、継続フォローアップ |
| **リスキリング** | 全職員のデジタル知識底上げ（人事課主導） |

### ⑤ デジタルを活用した職員エンゲージメント向上（P24）

| 取組 | 内容 |
|------|------|
| ワーク・エンゲイジメントの可視化 | デジタルツールで職員の熱意・満足度を測定・可視化 |
| **HRMシステム導入** | 人事情報を活用した適材適所の人員配置。2025年検討 → 2027年稼働 |

## 推進体制

```
デジタル・スマートシティ推進本部（本部長：市長）
  └ デジタル・スマートシティ推進課（事務局）
       └ 推進WG（プロジェクトチーム適宜設置）
            └ 各所属
```

- **外部人材活用**：浜松市フェロー（デジタル・データ活用の専門家）を積極活用
- **ICTガバナンス**：情報システムの企画・調達・運用に関する適正化ガイドライン（2022年4月策定）

## 杏林堂コミュニケーション基盤検討への示唆

### 浜松市との共通課題
| 課題 | 浜松市の対応 | 杏林堂への示唆 |
|------|------------|--------------|
| 場所を選ばない働き方 | 端末モバイル化・クラウドPBX | **MS365（Teams）+ モバイル端末**で同様の基盤構築可能 |
| 電話業務のデジタル化 | クラウドPBX導入 | 店舗・本部間の電話コミュニケーション見直しに参考 |
| AI活用推進 | 生成AI利用促進・AI議事録・AIシフト | **AIシフト作成**は杏林堂の労務課題（シフト最適化）と直結 |
| 人材育成体制 | DXコア・メンター・リスキリング3層 | CX推進部のDX人材育成ロードマップに応用可能 |
| HRMシステム | 人事情報で適材適所・エンゲージメント向上 | POSITIVE（共通導入済み）との連携で実現可能性あり |

### 浜松市が実践する「小さく始める」アプローチ
- 毎年度各所属で**1業務以上の業務改革を義務化**
- PoCを繰り返し、成功事例を横展開
- → 杏林堂でも**MS365配布（2027年2月）後、1部門1業務改革**から始めるアプローチが有効

### コミュニケーション基盤構築の優先順位（浜松市の順序から）
1. **NW・端末基盤整備**（モバイル化・クラウド接続）
2. **グループウェア・チャット統合**（Teams等）
3. **テレワーク・フリーアドレス制度化**
4. **AI活用**（議事録・シフト・生成AI）
5. **HRM/エンゲージメント可視化**
